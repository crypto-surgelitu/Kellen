# Catch My Love 💖

A premium, interactive web-based game where players catch falling hearts to accumulate points and progress through difficulty tiers. Built with the **Radiant Heart** design system for an editorial, mobile-first experience.

## ✨ Features
- **Dynamic Arcade Gameplay:** Catch three types of hearts (Normal, Golden, Heartbreak) as the game progressively intensifies.
- **Affection Explosion:** Real-time visual feedback with floating "Sweet Words" upon every successful catch.
- **Radiant Heart Design System:** A visual language defined by tonal layering, "No-Line" rules, and glassmorphism.
- **Responsive Controls:** Unified mouse and touch inputs with near-zero latency.
- **Hall of Love:** Global leaderboard tracking high scores across all romantic players.
- **Mobile Optimized:** Designed specifically for portrait engagement with high-end typography (Poppins, Inter, Be Vietnam Pro).

## 🛠️ Technical Stack
- **Core:** Vanilla JavaScript (ES Module-based architecture)
- **Styling:** Tailwind CSS v4 (Integrated with `@tailwindcss/vite`)
- **Physics & Motion:** Custom `requestAnimationFrame` loop + CSS Animations
- **Infrastructure:** Vite 8.x for ultra-fast HMR and bundling
- **Feedback:** Mobile Haptics API integration

## 🚀 Getting Started

### Prerequisites
- Node.js (Latest stable version)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/crypto-surgelitu/Kellen.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## 🎨 Design Principles
- **The "No-Line" Rule:** We use tonal layering and background shifts instead of borders for a more modern, premium look.
- **The "Soft Spring" Signature:** Every transition and animation follows a custom spring curve for a lively, organic feel.
- **Tonal Depth:** Utilizing a palette ranging from deep cranberry to soft rose to create a rich, layered UI.
